What does this do?

If your FXCommands buttons show as "action not found" after updating, your profile may reference an old plugin UUID. This tool fixes profiles from any previous version - including com.egg-rp.fxcommands, tf.josh.fxcommands.action, and other legacy variants - updating them to the current tf.josh.fxcommands UUID.

How to use

  1. In the Stream Deck app, right-click your profile and choose Export
  2. Drop the exported .streamDeckProfile file above
  3. Download the updated file and double-click to import it back
